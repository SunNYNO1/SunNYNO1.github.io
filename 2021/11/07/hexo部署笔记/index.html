hexo部署笔记

hexo部署笔记

hexo安装

  • 安装nodejs
    • node -v
    • npm -v
  • 安装cnpm
    • npm install -g cnpm –registry=https://registry.npm.taobao.org (-g 全局安装；cnpm是国内的镜像源)
    • cnpm -v（验证cnpm安装是否成功）
  • 安装hexo
    • cnpm install -g hexo-cli
    • hexo -v

搭建博客

  • 新建本地文件夹放置博客mybolg
  • 初始化博客 “hexo init”
  • 新建博客 “hexo n ‘博客名字’ “
  • 清理博客记录 “hexo clean”
  • 生车博客 “hexo g”
  • 在本地启动博客 “hexo s”

将博客部署到远程仓库（github）

  • 新建远程仓库

    • 仓库名必须为“github昵称.github.io”(ex:SunNYNO1.github.io)

  • 安装git部署插件

    • cnpm install –save hexo-deployer

  • 修改hexo的配置文件config.yml

    • 配置deployment
      type: git
      repo: https://github.com/SunNYNO1/SunNYNO1.github.io.git
      branch: master

  • 部署到远端

    • hexo d（需要验证：github username与password 或者是 personal access token）
    • 生成token方法（github设置–developer settings–personal access tokens– generate token–输入token备注“For Hexo Blog”–全部对勾–generate）
    • 在远端访问博客：SunNYNO1.github.io
